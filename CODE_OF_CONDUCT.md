# Code of Conduct

## Our Standard

ASTra MCP is a technical project. We keep it simple:

- **Be direct and constructive.** Critique code, not people.
- **Assume good intent.** Most contributors are learning or helping.
- **Stay on topic.** Issues and PRs are for the project, not general debate.
- **No harassment.** This includes any targeting based on identity.

## Scope

Applies to GitHub issues, PRs, discussions, and any space where you represent this project.

## Enforcement

Violations reported to [nsatyasaicharan@gmail.com](mailto:nsatyasaicharan@gmail.com).

Responses: warning → temporary ban → permanent ban, depending on severity.
